# PlayOptions

### Overview

| Options     | Type      | Description                                                   |
| ----------- | :-------- | ------------------------------------------------------------- |
| `endTime`   | `number`  | `The position to end the track.`                              |
| `noReplace` | `boolean` | `Whether to not replace the track if a play payload is sent.` |
| `startTime` | `number`  | `The position to start the track.`                            |
